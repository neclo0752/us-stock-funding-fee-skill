# Calculation Rules

## Variables
- `A`: 用户输入金额
- `C`: 用户输入币种（`USD` 或 `HKD`）
- `R`: 汇率（`USDHKD`），默认 `7.8`

换算：
- 若 `C = USD`：`A_usd = A`，`A_hkd = A * R`
- 若 `C = HKD`：`A_hkd = A`，`A_usd = A / R`

## Fee Formulas

### 方案一（港币直入嘉信花旗香港）
- 费率：`0.16%`
- 计算：`fee1_usd = A_usd * 0.0016`
- 等价：`fee1_hkd = A_hkd * 0.0016`
- 备注：嘉信端港币兑美元折损

### 方案二（美元电汇到嘉信花旗美国）
- 固定：`fee2_usd = 21.88`
- 等价：`fee2_hkd = 21.88 * R`
- 备注：固定手续费

### 方案三（港币到 wise 换美元再入金）
- 费率：`0.4%`
- 计算：`fee3_usd = A_usd * 0.004`
- 等价：`fee3_hkd = A_hkd * 0.004`
- 备注：wise 按比例收兑换费

### 方案四（美元到 wise 后入金）
- 固定：`fee4_hkd = 10`
- 等价：`fee4_usd = 10 / R`
- 备注：需 wise 港币账户，固定收费

### 方案五（汇丰香港到汇丰美国再入金）
- `fee5_usd = 0`
- `fee5_hkd = 0`
- 备注：需 HSBC US 账户

## Decision Points

### 方案一 vs 方案二临界点
- `21.88 / 0.0016 = 13,675 USD`
- 默认汇率 `R = 7.8` 时：`13,675 * 7.8 = 106,665 HKD`
- 规则：
  - 小于临界点：方案一更省
  - 大于临界点：方案二更省

### Q5 双换汇临界点（美元先换港币再走方案一）
- 总折损系数：`0.4% + (1 - 0.4%) * 0.16% = 0.55936%`
- 临界点：`21.88 / 0.0055936 = 3,911 USD`（约）
- 规则：
  - 小于 3,911 USD：可考虑该路径
  - 大于 3,911 USD：不建议

4. 补充关键临界点与一条风险提示（尤其是 wise 路径）。
